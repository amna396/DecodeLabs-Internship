# Project 2 - Data Classification Using AI

## Project Overview

This project was completed as part of my **DecodeLabs AI Internship**.

The goal of this project is to build a simple Machine Learning classification model using the famous Iris Flower Dataset. The model learns patterns from flower measurements and predicts the species of unseen flowers.

---

## Objectives

- Load and understand a dataset
- Split data into training and testing sets
- Train a Machine Learning model
- Predict flower species
- Evaluate model performance
- Visualize the dataset

---

##  Dataset

The project uses the **Iris Dataset** provided by Scikit-learn.

**Dataset Details**

- Total Samples: **150**
- Features: **4**
- Classes: **3**

### Features

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### Flower Species

- Setosa
- Versicolor
- Virginica

---

##  Machine Learning Algorithm

**Decision Tree Classifier**

##  Technologies Used

- Matplotlib
