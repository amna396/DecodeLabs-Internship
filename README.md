# UET Lahore Admission Chatbot (Fall 2026)

## Project Overview

This project is a Rule-Based AI Chatbot developed in Python using the Tkinter GUI library.

The chatbot assists prospective students by answering common questions related to UET Lahore Undergraduate Admissions (Fall 2026). It responds using predefined rules and keyword matching without using machine learning.

---

## Features

- User-friendly GUI built with Tkinter
- Rule-based conversation system
- Admission eligibility information
- Engineering programs
- Computing programs
- Business & Science programs
- Merit calculation
- Fee structure
- Departments
- ECAT Programs
- Non-ECAT Programs
- Admission Categories
- Admission Schedule
- Required Documents
- Application Guide
- Greeting and Exit Commands
